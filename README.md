# 🏦 Bank REST API

REST API для управления банковскими картами с аутентификацией JWT и документацией Swagger.

## ✨ Возможности

- 🔐 JWT аутентификация
- 💳 CRUD операции с банковскими картами
- 📚 Swagger UI документация
- 🗄️ PostgreSQL база данных
- 🐳 Docker поддержка

## 🚀 Быстрый запуск

### Docker
```bash
docker-compose up --build -d
```

## 🌐 API Endpoints

- **Swagger UI**: http://localhost:8080/swagger-ui.html
