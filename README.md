# Bank REST API — сервис банковских карт

REST API для управления банковскими картами: выпуск, блокировка, переводы между картами, аутентификация по JWT. Учебный проект (реализация типового тестового задания), но собран «по-взрослому» — слоёная архитектура, миграции, тесты и Swagger из коробки.

## Что умеет

- Регистрация и вход, **JWT** (access + refresh токены)
- Роли (админ / пользователь), доступ к операциям разграничен по ролям
- CRUD банковских карт, маскирование номера (`**** **** **** 1234`)
- Переводы между своими картами
- Swagger UI с описанием всех эндпоинтов

## Технологии

- **Java 17**, **Spring Boot** (Web, Security, Data JPA, Validation)
- **PostgreSQL**, миграции через **Liquibase**
- **JWT** (jjwt) поверх Spring Security
- **Swagger / OpenAPI** (springdoc)
- **Docker** Compose
- Тесты: **JUnit**, юнит + интеграционные

## Запуск

```bash
cp .env.example .env      # заполнить пароль БД и секрет JWT
docker compose up --build
```

Swagger UI: <http://localhost:8080/swagger-ui.html>

Секреты (пароль БД, ключ подписи JWT) читаются из переменных окружения — в коде и конфигах их нет.
